# 🌾 KrishiMitra UI

KrishiMitra is an AI-powered agricultural assistant designed to empower farmers with real-time data, crop insights, and voice-enabled support in multiple languages. This UI project simulates the mobile interface of the KrishiMitra platform, created using only HTML and CSS.

---

## 🚀 Features

- 🎤 **Voice Interface:** Interactive voice support for natural communication
- 🌦 **Live Weather Forecasting:** Display current and upcoming weather conditions
- 📷 **Crop Disease Diagnosis:** Upload images for AI-based disease detection with remedies
- 📈 **Market Price Tracker:** Live mandi rates with price trends and AI suggestions
- 🌱 **Smart Crop Planning:** Recommendations based on season, soil, and region
- 🏛️ **Government Schemes:** Display and apply for eligible agricultural schemes
- 🗺️ **Crop Health Mapping:** Visual health indicators of farms using a map layout
- 🌐 **Multilingual UI:** Supports English, Hindi, and Kannada navigation

---

## 🖥️ Tech Stack

- **HTML5** – Markup structure
- **CSS3** – Responsive styling, animations, and layout design
- **Emmet** – For rapid HTML authoring (during development in VS Code)
